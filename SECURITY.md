# Security Policy

## Reporting a vulnerability

Please report suspected vulnerabilities through GitHub Security Advisories for this repository. If private reporting is unavailable, open a private maintainer contact channel before disclosing details publicly.

When possible, include:

- affected version or commit hash
- reproduction steps
- expected impact
- any available mitigation notes

We will acknowledge receipt and follow up with a remediation timeline. Please avoid opening public issues for unpatched security defects.
